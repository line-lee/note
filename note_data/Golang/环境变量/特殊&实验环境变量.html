🔬 特殊 & 实验环境变量

⚠️ 注意：以下环境变量多为实验性质，在不同 Go 版本中行为可能变化，生产环境请谨慎使用。

GOEXPERIMENT

启用实验性功能，多个实验用逗号分隔。

export GOEXPERIMENT=loopvar,fieldtrack
实验标志说明引入版本
loopvar修复循环变量捕获问题（Go 1.22+ 默认行为）1.21
fieldtrack字段跟踪实验1.20
rangefuncrange over func 实验1.22

GOGC

控制垃圾回收的触发阈值。默认值为100，表示当新分配内存达到当前存活内存的100%时触发GC。

export GOGC=200   # 降低GC频率，但占用更多内存
export GOGC=off   # 关闭自动GC（Go 1.19+）

GODEBUG

调试和运行时行为控制，多个选项用逗号分隔。

export GODEBUG=gctrace=1,http2client=0
选项说明
gctrace=1打印GC日志
schedtrace=1000每1000ms打印调度器跟踪
http2client=0禁用HTTP/2客户端
inittrace=1打印包初始化跟踪
💡 小技巧：使用 go env -w GODEBUG=gctrace=1 可以永久设置环境变量。

GOARM / GOARM64

针对ARM架构的优化版本设置。

export GOARM=7      # ARMv7 优化
export GOARM64=v8.0 # ARM64 v8.0 指令集

GOMIPS / GOMIPS64

MIPS架构的浮点运算设置。

export GOMIPS=hardfloat   # 硬件浮点
export GOMIPS=softfloat   # 软件浮点

GO386

386架构的指令集选择（已废弃，推荐使用 GOAMD64）。

export GO386=sse2   # 使用SSE2指令集
