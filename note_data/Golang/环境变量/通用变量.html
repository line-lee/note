📦 Go 通用环境变量

📌 快速查看：运行 go env 查看所有当前环境变量设置。

GOROOT

Go 语言的安装根目录。通常不需要手动设置，Go 会自动检测。

export GOROOT=/usr/local/go
# Windows: set GOROOT=C:\Go

GOPATH

工作空间目录，包含 src、pkg、bin 三个子目录。Go 1.8+ 默认值为 $HOME/go。

export GOPATH=$HOME/go
# 多个路径用冒号分隔（Windows用分号）
export GOPATH=$HOME/go:$HOME/work/go
子目录用途
src/存放源代码
pkg/存放编译后的包文件
bin/存放编译后的可执行文件

GOBIN

指定 go install 安装可执行文件的目录。默认是 $GOPATH/bin。

export GOBIN=$HOME/bin

GOPROXY

Go Module 代理服务器地址，用于加速依赖下载。多个地址用逗号或竖线分隔。

# 使用国内代理（推荐）
export GOPROXY=https://goproxy.cn,direct

# 使用官方代理（需翻墙）
export GOPROXY=https://proxy.golang.org,direct

# 直连模式（不使用代理）
export GOPROXY=direct

# 关闭代理校验和检查
export GOSUMDB=off
🚀 国内加速：推荐设置 GOPROXY=https://goproxy.cn,direct

GOPRIVATE

指定私有仓库路径，不会通过代理下载，也不会进行校验和验证。

export GOPRIVATE=gitlab.mycompany.com,github.com/myprivate/*

GONOSUMDB / GONOPROXY

更精细的控制：

# 不进行校验和验证的模块
export GONOSUMDB=gitlab.mycompany.com/*

# 不使用代理的模块
export GONOPROXY=gitlab.mycompany.com/*

GOINSECURE

允许使用 HTTP 而非 HTTPS 拉取的模块路径。

export GOINSECURE=insecure.example.com

GOMODCACHE

Module 缓存目录，默认是 $GOPATH/pkg/mod。

export GOMODCACHE=$HOME/.cache/go-mod

GOCACHE

构建缓存目录，默认是 $HOME/.cache/go-build。

export GOCACHE=$HOME/.cache/go-build

GOENV

环境变量配置文件路径，默认是 $HOME/.config/go/env。

# 使用 go env -w 永久设置变量
go env -w GOPROXY=https://goproxy.cn,direct
go env -w GOPRIVATE=gitlab.mycompany.com

常用命令速查

# 查看所有环境变量
go env

# 查看单个变量
go env GOPATH

# 永久设置变量（推荐）
go env -w GOPROXY=https://goproxy.cn,direct

# 恢复默认设置
go env -u GOPROXY

# 查看当前模块信息
go list -m all
📖 推荐配置（~/.bashrc 或 ~/.zshrc）： 
export GOPROXY=https://goproxy.cn,direct
export GOPRIVATE=gitlab.mycompany.com
export GO111MODULE=on
